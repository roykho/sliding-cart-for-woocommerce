== Description ==

Adds a floating cart to your shop.

There is no active support for this plugin here.  Please go here to contribute https://github.com/roykho/floating-cart-for-woocommerce

== Installation ==

1. Be sure WooCommerce plugin is already installed.
2. Upload the folder `floating-cart-for-woocommerce` to the `/wp-content/plugins/` directory.
3. Activate the plugin through the 'Plugins' menu in WordPress.
4. Go to WooCommerce->settings->products tab->floating-cart section to set your settings.
